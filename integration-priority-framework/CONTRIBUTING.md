# Contributing

We welcome suggestions and improvements to the integration priority framework.

## Ways to contribute

- Open an issue to suggest improvements to the framework or scoring model.
- Submit a pull request with wording, examples, or clarity improvements.
- Share examples of how you adapted the framework for your team.

## Guidelines

- Keep contributions practical and specific.
- Preserve the core structure of the framework unless there is a strong reason to change it.
- Favor language that works for cross-functional product and engineering teams.
